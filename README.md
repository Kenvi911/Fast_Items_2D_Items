# Fast Items — 2D Items

> **1.21.11 port** of [Fast Items](https://modrinth.com/mod/fast-items) by [Noryea](https://modrinth.com/user/Noryea).
> All credit for the original mod goes to them.

Optimizes item rendering by transforming item entities into flat 2D sprites.

- For **2D items** (such as potions), this mod avoids rendering their sides, so they are completely *flat*
- For **3D items** (such as blocks), this mod makes them always face the camera — just like in Lunar Client / BLC

## Configuration

Use [ModMenu](https://modrinth.com/mod/modmenu) to access the in-game config screen.

| Option | Default | Description |
|---|---|---|
| **Enable this mod** | ON | Toggle the mod on/off |
| **Render entity shadows** | ON | Show shadows under dropped items |
| **Render sides of 2D items** | OFF | Show the sides of flat items |
| **Affects 3D models** | ON | Also apply to 3D items like blocks |
